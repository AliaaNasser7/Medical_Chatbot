# Smart Medical Chatbot

**Overview**  
This module provides an AI-powered chatbot capable of delivering accurate and efficient medical information. It leverages modern NLP tools to assist users with symptom analysis and general medical queries.

**Technologies Used**
- Python
- HuggingFace embeddings
- Faiss CPU for vector storage

**Features**
- Fast and reliable symptom analysis
- Efficient information retrieval
- Built with open-source AI tools

**Usage**
1. Set up dependencies as listed in requirements.txt.
2. Run the chatbot script and interact with the system through the terminal or integrated interface.
